## FAQ

**Does vpn proxy for privacy hide my IP?**  
Yes. Traffic routes through the VPN proxy so sites see the VPN endpoint, not your real IP.

**Will this slow my connection?**  
Some overhead is normal. Pick a nearby server first; use the global list only when you need a specific region.

**What does the Kill Switch actually do?**  
It blocks traffic if the VPN drops so apps don’t fall back to your normal connection and leak your IP.

**Is “No-Logs policy” guaranteed?**  
It’s a stated policy goal and a selection rule in this guide. Always read the provider terms on the download page before you rely on it.
